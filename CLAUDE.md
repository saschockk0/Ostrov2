# CLAUDE.md

## Проект

Одностраничный сайт Парусного Клуба «Остров» с ключевой целью: получить заявку через калькулятор и пошаговый опросник.

## Обязательные требования

- Язык интерфейса: русский.
- Визуальный стиль: минимализм, синие оттенки.
- Главный CTA: `Оставить заявку`.
- Выходные для расчета: пятница, суббота, воскресенье.
- Обязательные контактные поля: имя и телефон.
- Заявка должна:
  - сохраняться в БД,
  - отправляться на email менеджера,
  - иметь задел для отправки в Telegram-бота.

## Функциональность первой версии

- Hero-блок с видео.
- Блоки: лагерь, флот, мероприятия, мини-прайс, калькулятор, FAQ, медиа/отзывы, как добраться.
- Пошаговый опросник с условной логикой.
- Расчет стоимости по прайсу из `instructions.md`.
- Антиспам:
  - Cloudflare Turnstile (если задан ключ),
  - honeypot,
  - rate limiting.

## Технический стек (MVP)

- Frontend: HTML/CSS/Vanilla JS.
- Backend: Node.js + Express.
- БД: SQLite.
- Почта: Nodemailer (SMTP через env).

## Принципы реализации

- Логику цен хранить централизованно на backend.
- На клиенте показывать предварительный расчет, перед отправкой пересчитывать на сервере.
- Не отправлять заявку без обязательных полей и валидных дат.
- Сделать код максимально простым и читаемым, без лишних зависимостей.
