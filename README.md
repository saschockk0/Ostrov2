# Парусный Клуб «Остров» — MVP сайта

MVP включает:
- одностраничный сайт с блоками из ТЗ;
- быстрый калькулятор стоимости;
- пошаговый опросник заявки;
- backend API на Express;
- сохранение заявок в SQLite;
- отправку заявки на email менеджера (SMTP);
- антиспам через Turnstile + honeypot + rate limit.

## Запуск

1. Установите зависимости:

```bash
npm install
```

2. Создайте `.env` на основе `.env.example` и заполните SMTP/Turnstile.

3. Запустите проект:

```bash
npm start
```

4. Откройте:
[http://localhost:3000](http://localhost:3000)

## API

- `GET /api/config` — конфигурация фронтенда.
- `POST /api/quote` — расчет стоимости.
- `POST /api/applications` — сохранение и отправка заявки.

## Важно

- Если SMTP не настроен, заявка сохранится в БД, но email не отправится.
- Telegram-отправка подготовлена архитектурно как следующий шаг.
