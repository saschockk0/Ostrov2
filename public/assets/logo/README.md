# Логотип «Остров» — набор файлов

Единый фирменный знак парусного клуба. Все файлы векторные (SVG), с **встроенными
шрифтами** (рукописный «Остров» + Montserrat для «Парусный клуб»), поэтому отображаются
одинаково в браузере, в письме и при открытии напрямую — без подгрузки шрифтов.

Превью всех вариантов: **`/assets/logo/`** (страница `index.html`).

## Файлы

| Файл | Что это | Где использовать |
|---|---|---|
| `ostrov-logo-color.svg` | Полный логотип в цвете | Шапка сайта, светлый фон — основной вид |
| `ostrov-logo-white.svg` | Полный логотип, белая выворотка | Подвал, hero, тёмные секции, на фото |
| `ostrov-logo-mono.svg`  | Полный логотип в один цвет | Печать, ч/б, водяной знак (см. «Mono» ниже) |
| `ostrov-mark-color.svg` | Только знак (катамаран), цвет | Фавикон, аватар, превью в соцсетях |
| `ostrov-mark-white.svg` | Только знак, белый | Иконка на тёмном фоне |
| `ostrov-mark-mono.svg`  | Только знак, один цвет | Кнопки, бейджи, акценты |

## Цвета

| Элемент | HEX |
|---|---|
| Паруса | `#5B92BD` |
| Корпус / слово «Остров» | `#5BA13D` |
| Подпись «Парусный клуб» | `#5f6c78` |
| Mono по умолчанию | `#0e2f46` (фирменный тёмно-синий) |

## Mono (один цвет)

Mono-файлы используют `fill="currentColor"`. Перекрасить можно **только если SVG
вставлен инлайн** в HTML (или применён как CSS-маска) — тогда он берёт CSS `color`:

```html
<span style="color:#1b4f72">
  <!-- содержимое ostrov-mark-mono.svg вставлено инлайн -->
  <svg viewBox="60 20 280 175">…</svg>
</span>
```

При использовании через `<img src="ostrov-logo-mono.svg">` цвет **не** наследуется
от страницы — знак останется тёмно-синим (значение по умолчанию). Для произвольного
цвета берите инлайн-вставку; для готовых цветов — `*-color` / `*-white`.

## Как пересобрать

Все варианты генерируются из одного источника
(`logo.svg` — геометрия и разделение цветов, `logo-light.svg` — встроенные шрифты):

```bash
node scripts/build-logo-set.mjs
```

Скрипт идемпотентен. Менять цвета/кернинг — в `scripts/build-logo-set.mjs`.

## Что осталось (по желанию)

Растровые фавиконы (`favicon.ico`, `android-chrome-*.png`, `apple-touch-icon.png`)
сейчас сгенерированы из прежней версии. При необходимости их можно перевыпустить
из `ostrov-mark-color.svg` (нужен растеризатор: например, `sharp` или онлайн-сервис).
